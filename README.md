# Tcmp-CSharp-SDK
A C# SDK to communicate with a TappyUSB using the TCMP protocol.

## Install SDK into Visual Studio Project using Nuget

Nuget page: https://www.nuget.org/packages/TapTrack.Tcmp/

`PM> Install-Package TapTrack.Tcmp`
